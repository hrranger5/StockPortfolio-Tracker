Stock Portfolio Tracker (portfolio_tracker.py)

Goal

To calculate the total investment value of a stock portfolio based on manually defined (hardcoded) stock prices.

Key Concepts

This script uses dictionaries for stock prices, basic arithmetic for calculations, and the csv module for file handling.

How to Run

Save the code as portfolio_tracker.py.

Open your terminal or command prompt.

Run the script using the Python interpreter:

python main.py


Usage

The program will prompt you to enter a stock ticker (e.g., AAPL, TSLA).

It will then ask for the quantity (number of shares).

Type DONE when you are finished entering all your holdings.

The script calculates and displays the total portfolio value.
